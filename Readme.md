# Smart Quiz Game 🎮

## Description

A Python-based quiz game with timer, difficulty levels, and score tracking.

## Features

* Multiple choice questions
* Timer-based answering
* Random question selection
* Difficulty levels
* Score saving system

## How to Run

1. Install Python
2. Run:
   python main.py

## Project Structure

* main.py → Main program
* quiz_logic.py → Quiz logic
* questions.py → Question database
* utils.py → Timer & score saving
